# dynamic-form

Creating a dynamic form using Vue.js (Vue3, Options API)

# to set-up

please fork on clone the repository to your local machine
SSH, or HTTP is fine.

once you have done that,
cd into the project directory
run

```sh
npm install
```

and then to run locally,

```sh
npm run dev
```
